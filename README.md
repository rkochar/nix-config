# Nix Config

```sh
ln -s ~/nix-config/configuration.nix /etc/nixos/configuration.nix
ln -s ~/nix-config/hardware-configuration.nix /etc/nixos/hardware-configuration.nix
ln -s ~/nix-config/home.nix ~/.config/home-manager/home.nix
```
